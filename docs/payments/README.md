## 支付渠道

- **Stripe** — 全球用户（USD/EUR/GBP/JPY 等）
- **支付宝** — 中国大陆 CNY 用户
- **微信支付** — 中国大陆 CNY 用户

## 实施顺序

1. Phase 1：基础框架 + 数据模型
2. Phase 2：Stripe（全球优先）
3. Phase 3：支付宝
4. Phase 4：微信支付
5. Phase 5：运营完善

详见 [用户支付换积分改造方案.md](./用户支付换积分改造方案.md) 第 13 节。
